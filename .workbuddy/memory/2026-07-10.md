# 2026-07-10 工作日志

## 网站静态化改造
将 FoxMC（小狐狸生存服）Minecraft 服务器网站从 PHP+MySQL 动态站点转换为纯静态 HTML 站点。

### 改动内容
1. **新建 `data.js`** - 将所有 CMS 内容（站点设置、Hero、配置、帮助、特色、相册、团队、社区、页脚、公告、服务器状态、联系表单配置）从 PHP 数据库提取为静态 JS 对象。可编辑此文件来修改网站内容。
2. **重写 `script.js`** - 移除所有对 `admin/public_api.php` 和 `user/api/index.php` 的 fetch 调用：
   - UserSession: 改为始终显示"登录/注册"链接
   - Announce: 从 `data.js` 读取公告数据
   - CMS: 直接从 `window.__FOXMC_STATIC_DATA__` 读取所有内容
   - ServerStatus: 支持静态文案模式（data.js 配置）或可切换回 API 模式
   - ContactForm: 支持 mailto（默认）/ Formspree / 关闭三种模式
3. **修改 `index.html`** - 添加 `data.js` 引用，移除导航栏中指向 `user/panel.php?tab=shop` 的 PHP 商城链接
4. **保留 admin/ 和 user/ 目录** - 如果后续部署 PHP 后端，这些功能仍可恢复使用

### 使用方法
- 编辑 `data.js` 即可修改网站所有内容（IP 地址、Logo、文字、图片、团队等）
- 直接用任何静态文件服务器（Nginx/Apache/GitHub Pages）部署
- 联系表单默认使用 mailto 协议，可在 data.js 中切换为 Formspree

## 接入 MCSManager 开放 API（真实在线人数）
- 接口: GET /api/openapi/info → 返回 { online, players, port, motd, version, cpuCore, memUsage, ... }
- `data.js` 中配置 `server_status.mode = "api"`、`api_url = "https://panel.ambercat.top/api/openapi/info"`
- `script.js` 的 `serverStatus` 模块改造:
  - 调用 API 拉取 `online` 字段
  - 30 秒 TTL 缓存 + 周期刷新
  - 在线/离线状态点自动变色（绿/红）
  - API 失败时回退到 `static_text`/`static_value` 兜底文案
- Hero 区状态文字硬编码改为"加载中…"，由 JS 替换
- 注意: MCSManager 端需允许跨域（Access-Control-Allow-Origin），否则浏览器会拦截
